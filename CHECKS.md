## Not Checked — Requires Manual Testing

The following criteria cannot be reliably automated and require manual or screen reader testing:

- **2.4.7** Focus Visible — keyboard focus indicator is visible
- **3.2.3** Consistent Navigation — navigation is consistent across pages
- **4.1.3** Status Messages — live region announcements
- Actual keyboard trap testing
- Screen reader compatibility (NVDA, JAWS, VoiceOver)
- Touch target sizing (2.5.5)
- 35+ additional WCAG 2.1/2.2 criteria
